Keybindings
  • ⪡A⪢ - Add vertex
  • ⪡G⪢ - Generate new graph
  • ⪡R⪢ - Randomize placement
  • ⪡0⪢ - Fit to screen
    • —— 2D mode ——
  • ⪡-⪢ - Zoom out
  • ⪡=⪢ - Zoom in
  • ⪡ ⪢ - Pause animation
  • ⪡←⪢ - Move left
  • ⪡→⪢ - Move right
  • ⪡↑⪢ - Move up
  • ⪡↓⪢ - Move down

Troubleshooting

Disconnected graph with bipartite algorithm

Solution: Increase V set size.

Invalid scale free exponent with preferential selection with clustering algoritm

Solution: Keep the node degree ≤ 3.
