Keybindings:

  • A - Add Vertex
  • G - Generate New Graph
  • R - Randomize Placement
  • 0 - Fit To Screen
  • Minus(-) - Decrease View Scale
  • Equals(=),Plus(+) - Increase View Scale
  • (Space Bar) - Pause Animation
  • Left Arrow - Move Left
  • Right Arrow - Move Right
  • Up Arrow - Move Up
  • Down Arrow - Move Down

Troubleshooting

Q: Disconnected graph with bipartite algorithm
A: Increase V set size

Q:Not obtaining a valid scale free exponent with preferential selection with clustering algoritm
A: Keep the node degree <= 3
